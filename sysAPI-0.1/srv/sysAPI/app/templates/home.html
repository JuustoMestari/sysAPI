/cat/{path}

Retrieve file content. Retrieve only first 50kB of utf8 readable files.
Response for
/cat/home/admin/hello.txt
{
  "file": {
    "filecontent": "Hello World!\n",
    "filelimit": 50000,
    "filepath": "/root/hello.txt",
    "filesize": 13
  }
}
